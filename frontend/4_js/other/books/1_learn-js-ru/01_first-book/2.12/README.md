# Операторы нулевого слияния ?? и нулевого присваивания ??=

## ??

Значение определено, если оно не **null** и не **undefined**

Если **a** определено, то **a**, иначе **b**
```javascript
a ?? b;
```

Это удобный синтаксис, как получить значение, которое определено

То есть в развёрнутом виде нулевое слияние это:
```javascript
result = (a !== null && a !== undefined) ? a : b;
```

?? используют, чтобы задать значение по умолчанию для неопределённой переменной
```javascript
let user;
alert(user ?? "Аноним"); // > Аноним
```

Вот так можно показать первое значение, которое определено:
```javascript
let fn = null;
let ln;
let nn = 'superuser';
alert(fn ?? ln ?? nn ?? 'admin');
```

### Сравнение в ИЛИ

?? работает как || , только ИЛИ возвращает первое истинное значение, а нулевое слияние первое определённое значение

ИЛИ не различает значения (все они false):
- false
- 0
- ""
- null
- undefined

На практиче часто нужно использовать значение по умолчанию только тогда, когда там null или undefined
```javascript
let height = 0;
console.log(height || 100); // 100
console.log(height ?? 100); // 0
```

### У ?? приоритет как и у ||

Использование ?? вместе с && или || может вызвать ошибку если не использовать скобки
```javascript
let x = 1 && 2 ?? 3; // SyntaxError
console.log(x);
```

А вот так со скобками ошибки не будет:
```javascript
let x = (1 && 2) ?? 3;
console.log(x);
```

## ??=

Можно проверить равна ли переменная null/undefined и если да - присвоить переменной значение

Можно так:
```javascript
let userAge = null;
if (userAge === null || userAge === undefined) {
  userAge = 18;
}
```

А сокращённо можно так:
```javascript
// x ??= y
let uAge = null;
uAge ??= 18;
console.log(uAge); // 18
```
