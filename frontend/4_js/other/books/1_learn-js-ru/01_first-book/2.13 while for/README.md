# Циклы

Повторять однотипные действия

## while

Цикл ниже выводит значение i, пока i меньше 3:
```javascript
let i = 0;
while (i < 3) {
  console.log(i);
  i++;
}
```

Одно выполнение цикла - это **итерация**

Иногда можно и нужно сделать "вечный" цикл, когда условие для прекращения цикла не наступает
