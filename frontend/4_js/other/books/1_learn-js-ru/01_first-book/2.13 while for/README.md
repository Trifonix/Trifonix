# Циклы

Повторять однотипные действия

## while

Цикл ниже выводит значение i, пока i меньше 3:
```javascript
let i = 0;
while (i < 3) {
  console.log(i);
  i++;
}
```

Одно выполнение цикла - это **итерация**

Иногда можно и нужно сделать "вечный" цикл, когда условие для прекращения цикла не наступает

## do while

Этот цикл делает тело (действия) минимум 1 раз, а потом столько, пока будет выполняться условие в скобках

```javascript
let i = 5;
do {
  console.log('i =', i);
  i--;
} while (i !== 0);
```

## for

Это такой цикл который состоит из трёх штук: итератор, условие и действие. Например, задаётся переменная со значением 0, действием она увеличивается на 1 каждую итерацию и если в конце условие выхода из цикла выполняется, то будет прерыванием цикла (иначе следующая итерация)

```javascript
for (let x = 0; x <= 10; x++) {
  console.log('Привет');
  console.log(`Сейчас x равно ${x}`);
}
```

Как части оператора for ;; всегда должны быть в скобках

Можно заранее выйти из цикла for ключевым словом break (при условии, например)

Есть ещё ключевое словое continue для того, чтобы завершить текущую итерацию и начать новую

Операторы break/continue не могут использоваться в тернарнике (будет синтаксическая ошибка)

Перед циклом можно ставить метку, типа 'label' и использовать break/continue для перехода между уровнями вложенных циклов

```javascript
let y = 10;
console.log('заходим');
firstlabel: for (; y > 0; y--) {
  console.log('итерация первого цикла №', y);
  for (let j = 0; j <= 5; j++) {
    if (j === 4 && y === 6) break firstlabel;
  }
}
console.log('вышли');
```
