# Логические операторы

В джаваскрипте 7 логический операторов.

Вот 5 логических операторов:
- ИЛИ ||
- ИЛИ-присваивание ||=
- И &&
- И-присваивание &&=
- НЕ !

B ещё 2 логических оператора:
- нулевое слияние ??
- нулевое присваивание ??=

## ИЛИ ||

Двойной символ вертикальной черты

Обычно в программировании ИЛИ используют для булевых значений (если одно true, то true, иначе false)

В js по-другому

Если рассматривать булевые значения, то при любом true -> вернётся true

Значения не логического типа приводятся к логическому

Число 1 - это true

Число 0 - это false

```javascript
// тут 1 -> true, 0 -> false
if (1 || 0) {
  alert('истинно!');
}
```

ИЛИ используют, что проверять условия в скобках (ИЛИ-ИЛИ)

### ИЛИ находит первое true

Дополнительные возможности ИЛИ в js

```javascript
result = value1 || value2 || value3;
```

ИЛИ вычисляет операнды слева направо, конвертирует в булевое. Если true, ИЛИ останавливается и возвращает значение этого операнда. Если все были false, тогда возвращается последний из операндов

```javascript
alert( 1 || 0 ); // вернёт 1
alert( true || 'какая-то строка' ); // вернёт true
alert( null || 1 ); // вернёт 1
alert( null || 0 || 1 ); // вернёт 1
alert( undefined || null || 0 ); // вернёт 0
```

Допустим, если несколько переменных: firstName, lastName, nickName. Все необязательные. Если между всеми поставить ИЛИ и последним ИЛИ сделать строку, то при всех false вернётся последнее значение - строка

```javascript
let fn = "";
let ln = "";
let nn = "";
alert(fn || ln || nn || "text");
```

### Сокращённое вычисление ИЛИ

ИЛИ обрабатывает операнды и возвращает первое значение true или выражение

```javascript
true || alert('no');
false || alert('yes');
```

Иногда это используется если условие слева false (чтобы вернуть что-то справа или вызвать)

## ИЛИ-присваивание ||=

Этот оператор делает следующее:
- вычисляет выражение справа
- приводит значение слева к булевому типу
- присваивает переменной слева значение справа

В полном виде это было бы так:
```javascript
a || (a = b);
```

То есть это какбы проверка на true:
```javascript
let johnHasCar;
johnHasCar ||= "John has not car";
alert(johnHasCar);
```

Оператор ИЛИ-логическое - это синтаксический сахар, чтобы упростить запись if с приваиванием

## И &&

Обычно И возвращает true когда оба аргументы истинны, иначе false

```javascript
let hour = 12;
let minute = 42;
if (12 === hour && 42 === minute) {
  console.log('It\'s 12:42');
}
```

### И находит первое false

И делает так:
- вычисляет операнды слева направо
- каждый операнд в булевое (если false -> возврат его значения)
- если все true -> возрат последнего значения

И возвращает первое false, а ИЛИ - первое true

```javascript
console.log(null && 5) // null
console.log(0 && "text"); // 0
```

Приоритет у И больше, чем у ИЛИ

С оператором И можно немного упростить if-конструкцию
```javascript
let x = 42;
if (x > 0) console.log('more than 42');
x > 0 && console.log('more than 42 too');
```

Но улучше не сокращать if через И

## И-присваивание

Принцип И-присваивания похож на ИЛИ-присваивание

И-присваивание сработает только когда выражение слева true

```javascript
let greeting = "Hello";
greeting &&= greeting + " World!";
console.log(greeting);
```

И-присваивание тоже используют редко

## НЕ !

Оператор не инвертирует значение на другое булевое

```javascript
console.log(!true); // false
```

Бывает двойное НЕ (два !!)

Двойное НЕ - это аналог функции Boolean(), чтобы привести значение к логическому типу

```javascript
console.log(!!42); // true
```

У НЕ самый высокий приоритет
