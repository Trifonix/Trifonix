# Условия if

Действия при условиях

Используют if и ?

Пишется if потом скобки ()

Если выражение в скобках истина (true), тогда выполняются действия в фигурных скобках

```javascript
let year = prompt('year of ECMAScript-2015?');
if (2015 === +year) {
  alert('yes');
}
```

Действий может быть много

Лучше всегда использовать фигурные скобки для читаемости кода

# В логический тип

Выражение в скобках преобразуется к логическому типу

False будут 5 значений: 
- число 0
- пустая строка ""
- null
- undefined
- NaN

Остальные значения будут true

То есть, если после if в скобках (false), то действия не выполняются

Можно предварительно присвоить переменное логическое значение. И если передать его в if -> действия выполнятся

```javascript
const year = '2000';
let condition = (2000 == year); // true
if (condition) {
  alert('mm, good');
}
```

## блок else

После if может быть else (иначе)

else использовать не обязательно

Его суть - выполнить условия если в if в скобках (false)

```javascript
if (false) {
  alert('mm, good');
} else {
  alert('ou, bad!');
}
```

## else if

Может быть дополнительный if после основного if -> else if (иначе если) - если не выполняется основное условие

```javascript
let age = 11
if (age == 10) {
  alert(10);
} else if (age == 11) {
  alert(11);
} else {
  alert("all other");
}
```

## оператор ?

Это тернарный оператор, синтаксический сахар языка, чтобы сделать код короче

Этот способ записи сокращает большой if

Работает это так: проверяется значение выражение, если результат true, то выполняется первое значение после ? иначе второе

```javascript
let name = 'Neo';
let result;
result = (name == 'Smith') ? 'You are agent!' : 'You are Neo!';
alert(result);
```

Скобки вокруг проверочного выражения можно не писать, но лучше использовать для визуального восприятия

## несколько тернарников

Можно использовать тернарник в тернарнике

```javascript
let city = prompt('Город?', "Москва");
let result = (city == "Москва") ? 'Вы в Москве' :
  (city == "СПб") ? 'Вы в СПб' :
  (city == "Екат") ? 'Вы в Екате' :
  'Странный город';
alert(result);
```

## ещё один вариант использования ?

Можно вызывать функции в тернарнике

```javascript
let os = 'linux';
(os === 'windows') ? 
  alert("bad user") : alert("good user");
```

Использовать тернарник так не рекомендуется, лучше через обычный if
