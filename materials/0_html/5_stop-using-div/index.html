Перестаньте юзать дивы!

Вы всё ещё оборачиваете весь контент страницы в "дивы"? Остановитесь!

Есть лучший способ - использовать семантическую разметку.

Это улучшает доступность, повышает SEO и читабельность.

Проблема с чрезмерным использованием div в следующем.

Поисковики и программы чтения с экрана не понимают div.

Скринридеры игнорируют аттрибут ролей 'role'.

Много подряд идущих блоков div - не семантично!

Такой подход ни о чём не говорит поисковым роботам.

Решение - использовать семантические теги.

Они описывают цель - улучшаю доступность и SEO.

Тег header

для заголовков

Тег nav

для навигационного меню

Тег main

для основного содержимого страницы

Тег section

для группировки связанного содержимого

Тег article

самодостаточный контент новостей, постов

Тег aside

для бокового меню и обособленного содержимого

Тег footer

для нижней части сайта - подвал
Используйте div только тогда, когда нет подходящего семантического тега!
